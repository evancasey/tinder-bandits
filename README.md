tinder-bandits
==============

Hacking Tinder with Multi-armed Bandits

## Setup

Grab your Facebook oauth token off by visiting this [link](https://www.facebook.com/dialog/oauth?client_id=464891386855067&redirect_uri=https://www.facebook.com/connect/login_success.html&scope=basic_info,email,public_profile,user_about_me,user_activities,user_birthday,user_education_history,user_friends,user_interests,user_likes,user_location,user_photos,user_relationship_details&response_type=token) and grabbing the auth token out of the URL you are redirected to.

Run:
```
$ sbt run <your-facebook-id> <your-facebook-token>
```
